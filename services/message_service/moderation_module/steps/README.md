# Шаги модерации

## BanWordsStep - проверка запрещенных слов
Проверяет текст на наличие запрещенных слов и возвращает ModerationReason при обнаружении.

### Параметры
`forbidden_words: list` - список запрещенных слов

### Результаты 
`ModerationReason` - если найдено запрещенное слово
`None` - если текст без запретов

## NormalizationStep - нормализация текста
Проверяет текст на наличие избыточных пробелов, табуляций и переносов строк.
### Параметры
`MAX_SPACES_SEQUENCE` - максимальное допустимое количество пробелов подряд
`MAX_NEWLINES_SEQUENCE` - максимальное допустимое количество переносов строк подряд

### Результаты 
`ModerationWarning` - если обнаружены проблемы с форматированием: 
- `SPACES` - слишком много пробелов или табуляций
- `NEWLINES` - слишком много переносов строк
`None` - если текст нормализован

##  PIIStep - Поиск персональных данных
Обнаруживает персональные данные в тексте: номера телефонов и email-адреса.

### Параметры
`TEL_REG` - регулярное выражение для поиска российских телефонных номеров
`EMAIL_REG` - регулярное выражение для поиска email-адресов

### Результаты 
`ModerationWarning` - если найдены персональные данные:
- `PII_PHONE` - обнаружен номер телефона
- `PII_EMAIL` - обнаружен email-адрес
`None` - если персональные данные не найдены
